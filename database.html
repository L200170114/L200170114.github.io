Perintah Dasar DDL

Perintah Fungsi
Create Untuk membuat database, tabel, view dan kolom baru
Alter Untuk mengubah struktur tabel yang telah dibuat
Rename Untuk mengubah nama objek
Drop Untuk menghapus database dan tabel

Perintah Dasar DML

Perintah Fungsi
Update Untuk memperbaharui data lama menjadi data terkini
Insert Untuk menyisipkan atau memasukkan data baru kedalam tabel
Select Untuk mengambil data atau menampilkan data dari satu tabel atau
beberapa tabel dalam relasi
Delete Untuk menghapus data dari tabel
